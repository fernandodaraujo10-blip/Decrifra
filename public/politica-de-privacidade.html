Política de Privacidade

Vigente a partir de 27 de abril de 2026

O Decifra é um aplicativo de interpretação inteligente de filmes, séries, músicas e livros, desenvolvido por Fernando Araújo Ferreira Taise. Esta Política de Privacidade explica como coletamos, usamos, armazenamos e protegemos seus dados pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018).

Ao utilizar o Decifra, você concorda com as práticas descritas neste documento. Se não concordar, não utilize o app.

1. Dados que Coletamos

Coletamos os seguintes dados quando você usa o Decifra:

Dados de conta (ao fazer login com Google):
  • Nome completo
  • Endereço de e-mail
  • Foto de perfil (URL pública do Google)
  • Identificador único do Google (UID)
Dados de uso do aplicativo:
  • Histórico de pesquisas realizadas no app
  • Tipo e quantidade de análises realizadas
  • Plano de assinatura ativo (gratuito, Pro ou Premium)
  • Contadores de uso diário e mensal
  • Conteúdos salvos como favoritos
Dados técnicos:
  • Identificador anônimo do dispositivo (gerado localmente)
  • Tipo de dispositivo e sistema operacional (via Firebase Analytics)
  • Dados de sessão para funcionamento do serviço

2. Como Usamos seus Dados

Utilizamos suas informações para as seguintes finalidades:

  • Autenticação: identificar sua conta e manter sua sessão ativa
  • Personalização: exibir seu plano, histórico e favoritos
  • Controle de limites: aplicar as cotas de análises conforme o plano contratado
  • Assinaturas: processar e verificar pagamentos de planos
  • Melhoria do produto: entender quais funcionalidades são mais usadas
  • Comunicações: enviar notificações relacionadas ao serviço (somente com seu consentimento)

Não vendemos seus dados a terceiros. Não usamos suas informações para publicidade de terceiros.

3. Compartilhamento com Terceiros

Seus dados podem ser processados pelos seguintes prestadores de serviço, exclusivamente para viabilizar o funcionamento do Decifra:

Google Firebase (Google LLC)
Autenticação (Firebase Auth), banco de dados (Firestore), hospedagem e análises.
Política de privacidade do Firebase
Google Gemini API (Google LLC)
Processamento de IA para geração das análises de filmes, séries, músicas e livros.
Termos do Gemini API
TMDB — The Movie Database
Busca e metadados de filmes e séries (título, pôster, data).
Política do TMDB
OMDb API
Fonte secundária de metadados de filmes.
Site do OMDb

Todos esses serviços operam com suas próprias políticas de privacidade. As suas chaves de API são mantidas em servidores seguros e nunca expostas ao usuário final.

4. Armazenamento e Segurança

Seus dados de conta são armazenados no Google Firestore, com servidores localizados no Brasil (região southamerica-east1). O acesso a esses dados é protegido por:

  • Autenticação obrigatória via Firebase Auth
  • Regras de segurança no Firestore: cada usuário só pode ler e escrever seus próprios dados
  • Comunicações criptografadas via HTTPS/TLS
  • Chaves de API protegidas via Secret Manager do Google Cloud

Dados de pesquisa e cache são armazenados localmente no seu dispositivo (localStorage) e nunca enviados a servidores de terceiros sem necessidade.

5. Retenção de Dados

Mantemos seus dados enquanto você tiver uma conta ativa no Decifra. Ao encerrar sua conta, seus dados pessoais serão excluídos em até 30 dias, exceto quando houver obrigação legal de retenção.

Dados anonimizados e agregados (ex: estatísticas de uso) podem ser mantidos por prazo indeterminado, pois não permitem identificação pessoal.

6. Seus Direitos (LGPD)

Como titular de dados pessoais, você tem os seguintes direitos garantidos pela LGPD:

  • Acesso: saber quais dados temos sobre você
  • Correção: solicitar a atualização de dados incorretos
  • Exclusão: pedir a remoção dos seus dados
  • Portabilidade: receber seus dados em formato estruturado
  • Revogação de consentimento: retirar autorização dada anteriormente
  • Oposição: opor-se ao tratamento de dados em certas situações

Para exercer qualquer desses direitos, envie um e-mail para: familiaaraujoft@gmail.com

7. Cookies e Armazenamento Local

O Decifra utiliza localStorage do navegador (não cookies tradicionais) para armazenar preferências, tema, idioma e cache de buscas. Esses dados permanecem no seu dispositivo e não são transmitidos automaticamente.

O Firebase pode usar cookies técnicos para manter sessões de autenticação, conforme sua própria política.

8. Menores de Idade

O Decifra não é direcionado a menores de 13 anos. Não coletamos conscientemente dados de crianças. Se você acredita que uma criança forneceu dados ao app, entre em contato conosco para que possamos removê-los.

9. Alterações nesta Política

Podemos atualizar esta Política periodicamente. Quando houver mudanças significativas, notificaremos você dentro do aplicativo. Recomendamos revisar este documento regularmente.

O uso continuado do Decifra após alterações implica aceitação da nova política.
