Política de Privacidade

Última atualização: 28 de abril de 2026

Esta Política de Privacidade descreve como o Decifra coleta, utiliza e protege suas informações pessoais, em conformidade com a Lei Geral de Proteção de Dados — LGPD (Lei nº 13.709/2018).

1. Controlador dos Dados

Fernando Araújo Ferreira Taise
E-mail: familiaaraujoft@gmail.com
Site: https://decifra.app

2. Dados Coletados

Coletamos apenas o mínimo necessário para o funcionamento do aplicativo:

  • Nome e e-mail — obtidos via autenticação pelo Google (Google Auth).
  • Histórico de interpretações — textos, filmes, livros ou músicas que você analisou dentro do Decifra.

3. Finalidade do Uso

Seus dados são utilizados exclusivamente para:

  • Identificar e autenticar você no aplicativo.
  • Armazenar e exibir seu histórico pessoal de interpretações.
  • Gerenciar seu plano de assinatura.

4. Compartilhamento com Terceiros

Para oferecer o serviço, utilizamos os seguintes provedores externos, cada qual com sua própria política de privacidade:

  • Firebase / Google — autenticação e banco de dados.
  • Gemini API (Google) — processamento de inteligência artificial.
  • TMDB (The Movie Database) — metadados de filmes e séries.
  • OMDb API — informações complementares de filmes.
  • Mercado Pago — processamento de pagamentos.

Não vendemos nem cedemos seus dados a terceiros para fins de marketing.

5. Pagamentos

As transações financeiras são integralmente processadas pelo Mercado Pago. O Decifra não armazena dados de cartão de crédito, débito ou qualquer outro instrumento de pagamento.

6. Retenção de Dados

Seus dados são mantidos enquanto sua conta estiver ativa. Após uma solicitação de exclusão, os dados serão removidos em até 30 dias corridos.

7. Seus Direitos (LGPD)

Conforme a LGPD, você tem direito a:

  • Acesso — saber quais dados temos sobre você.
  • Correção — solicitar a atualização de dados incorretos.
  • Portabilidade — receber seus dados em formato estruturado.
  • Exclusão — solicitar a remoção de seus dados.
  • Revogação do consentimento — cancelar o uso de seus dados a qualquer momento.

Para exercer qualquer desses direitos, envie um e-mail para familiaaraujoft@gmail.com.

8. Cookies e Armazenamento Local

Podemos utilizar armazenamento local no navegador (localStorage) para manter sua sessão ativa e preferências de uso. Nenhum cookie de rastreamento ou publicidade é utilizado.

9. Segurança

Adotamos boas práticas de segurança, incluindo comunicação criptografada (HTTPS) e controle de acesso via Firebase Authentication. No entanto, nenhum sistema é 100% inviolável.

10. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Notificaremos sobre mudanças relevantes no próprio aplicativo ou por e-mail. A data de vigência está sempre indicada no topo deste documento.
